# Symptom & Impact Diary PWA v2

Upload all extracted files and the `icons` folder to the root of the existing GitHub repository and commit the changes.

The revised version includes a dashboard, bottom navigation, diary search and filters, calendar view, monthly statistics, significant-event flags, CSV/PDF/JSON tools, offline use, and an iPhone date/time width fix.

After updating GitHub Pages, close and reopen the installed app. If the old version remains cached, remove the Home Screen app, refresh the site in Safari, and add it to the Home Screen again.
