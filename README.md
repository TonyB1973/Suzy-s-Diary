# Symptom & Impact Diary PWA

A private, offline-first diary for recording symptoms and the effect they have on work, travel, eating, social plans and daily life.

## Features

- Quick dated diary entries
- Category and impact level
- Medication, food avoidance, location and delay fields
- Daily-life impact tick boxes
- Search and filters
- Edit and delete entries
- Printable report / Save as PDF
- CSV export
- JSON backup and restore
- Works offline after the first visit
- Data remains in the browser on the current device

## Upload to GitHub Pages

1. Create a new GitHub repository.
2. Upload every file and folder from this package to the repository root.
3. Open **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the **main** branch and **/(root)** folder, then Save.
6. GitHub will display the site address after deployment.

## iPhone installation

1. Open the GitHub Pages address in Safari.
2. Tap the Share button.
3. Choose **Add to Home Screen**.
4. Open the new icon from the Home Screen.

## Privacy

Entries are saved with `localStorage` in the browser. This app does not send the diary to a server.
A browser reset, app removal, device loss or site-data deletion can erase the records. Use **Backup JSON** regularly.
