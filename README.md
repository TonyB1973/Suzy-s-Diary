# Suzy's Diary v3

This package replaces the broken previous upload.

## Update instructions

1. Extract this ZIP.
2. Copy all files and the `icons` folder into the existing Working Copy repository.
3. Choose Replace when prompted.
4. Delete any duplicate `-2` files if they still exist.
5. Commit and push.
6. Open the site in Safari using `?v=3` once after deployment.

The old service worker is intentionally unregistered by this version to prevent stale cached files.
